# Database-Normalization-Analytics
# Database Schema Design and Normalization for Analytics-Oriented Applications

## Project Description
This project demonstrates database normalization from UNF to 3NF using a Retail Analytics example.

## Tools Used
- XAMPP (MySQL)
- phpMyAdmin
- GitHub

## Database Tables
- Customer
- Orders
- Order_Details
- Product
- Category

## Features
- Fully normalized schema (3NF)
- Foreign key relationships
- Analytical SQL queries
- Sample data insertion
